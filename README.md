# JavaScript-Fundamental
## Background
- This repository tracks the progress of certification tasks and labs provided via FreeCodeCamp's JS and Frontend Developement Libraries Certification.
- This course teaches you core JavaScript programming concepts such as working with variables, functions, objects, arrays, and control flow.
- It also talks about how to manipulate the DOM, handle events, and apply techniques like asynchronous programming, functional programming, and accessibility best practices.

## Languages Used
- Javascript and related libraries
